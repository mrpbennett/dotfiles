## Install

1. Ensure your system is up to date

- Ubuntu

```
sudo apt-get update
```

- CentOS

```
sudo yum upgrade
```

- MacOS

```
sudo softwareupdate -i -a
xcode-select --install
```

2. Clone with Git

```
git clone git@github.com:mrpbennett/dotfiles.git ~/.dotfiles
```

3. Run the installation script

```
source ~/.dotfiles/install.sh
```
